# 2025_MatchaTonic
2025컴퓨터공학과졸프

> **컴퓨터공학과 캡스톤디자인A / 졸업프로젝트**  
> **Promate**:AI가 주도하는 프로젝트 자동 세팅 & 협업 지원 플랫폼
> 
> ## 프로젝트 개요  
**MatchaTonic**dml Promate는 대학생 및 소규모 팀을 위한 AI 기반 올인원 협업 자동화 플랫폼입니다. <br>
팀 프로젝트 시작 시 반복되는 준비를 위한 준비(초기 세팅)의 번거로움을 제거하고,<br> AI가 분석한 최적의 일정과 리소스를 제공하여 팀이 프로젝트의 본질에 즉시 집중할 수 있도록 돕습니다. 

AI_repo : https://github.com/y3onk/2025_MatchaTonic_AI  <br>
FE+repo : https://github.com/sua005/2025_MatchaTonic_FE

### 기획 배경
- **협업 오버헤드 해결:** 노션 세팅, 깃허브 초기화, 규칙 제정 등 반복적인 초기 세팅 소요 시간 단축
- **일정 관리의 어려움:** 역할 분담과 타임라인 설계가 미숙한 초보 팀을 위한 가이드라인 필요
- **경험의 자산화:** 일회성으로 끝나는 프로젝트 경험을 지속 가능한 포트폴리오로 관리

## 핵심 기능

### 1.  초기 프로젝트 구조 자동 생성
사용자의 자연어 입력을 분석하여 프로젝트의 기틀을 단 몇 초 만에 구축합니다.
- **채팅형 인터페이스:** "대학생 4명이서 맛집 추천 웹/앱을 만들 거야"와 같은 자연어 요구사항 분석
- **워크스페이스 자동 구축:**
    - **Notion:** 회의록, 칸반 보드, 업무분담, 규칙 페이지가 포함된 템플릿 자동 생성 및 API 연동

### 2. AI Scheduler
프로젝트 목표와 마감일을 기반으로 실행 가능한 수준의 Task를 설계합니다.
- **Role-Based Task 분배:** 기획, 디자인, FE, BE 등 역할별 핵심 과업(WBS) 자동 생성
- **타임라인 최적화:** 전체 기간을 고려하여 마일스톤(중간 목표) 설정 및 주차별 할당
- **유기적 연동:** 생성된 일정은 Notion 캘린더/데이터베이스 및 Google Calendar에 즉시 반영

### 3. 지속적 관리 & 포트폴리오
단순 도구를 넘어, 사용자의 프로젝트 이력을 관리하는 커리어 플랫폼으로 기능합니다.
- **마이페이지 대시보드:** 진행 중인 프로젝트 현황(진행률) 및 최근 작업 조회
- **자동 아카이빙:** 완료된 프로젝트는 포트폴리오 카드 형태로 자동 변환 및 저장
- **상세 이력 관리:** 다른 포트폴리오 제작할 필요없이 프로젝트별 기여도, 사용 기술 스택, 맡은 역할이 정리된 상세 페이지 제공

### 4. 외부 서비스 심리스 연동
- **Notion:** 페이지 및 데이터베이스 양방향 동기화
- **GitHub:** 리포지토리 생성 및 웹훅(Webhook) 기반 이벤트 추적
- **Discord/Slack:** 주요 알림 및 회의록 요약 봇 연동
