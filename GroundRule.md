# Team Ground Rule  
> 팀원 협의에 따라 작성한 기본 규칙

---

## 1. 회의 및 일정
- 주 1회 정기 회의(11:00-12:00)
- 회의록은 Notion에 기록

## 2. 개발 원칙
- GitHub Flow 방식 사용 (main 브랜치 보호)
- 커밋 메시지는 영어 + 기능 단위 작성
- 코드 리뷰는 최소 1명 이상 확인 후 머지

## 3. 의사소통
- 카카오톡 단체방
- 중요한 결정은 노션 회의록에 남기고 전원 합의

## 4. 역할 분담
- 팀원 간 역할은 유연하게 조정 가능
- 특정 기간에 과부하 시 서로 지원

## 5. 기타
- 학업·개인 일정은 최소 1주일 전 공유
- 분쟁 발생 시 투표 + 멘토 교수님 자문
